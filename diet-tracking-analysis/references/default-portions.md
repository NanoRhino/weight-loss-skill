# Default Portions

Standard single-serving reference values. Use when user doesn't specify portion size — prefix with `~` in output.

| Food | Default Portion |
|------|----------------|
| Rice bowl (米饭) | ~150g |
| Egg (鸡蛋) | ~50g |
| Milk cup (牛奶) | ~250ml |
| Vegetable plate (蔬菜) | ~200g |
| Bread slice (面包片) | ~35g |
| Chicken breast (鸡胸肉) | ~120g |
| Noodles (面条) | ~200g |
| Tofu (豆腐) | ~150g |
| Apple (苹果) | ~200g |
| Banana (香蕉) | ~120g |

This list will grow over time. For foods not listed here, use USDA FoodData Central standard serving sizes or regional equivalents.
